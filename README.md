# Ede Marketing Website

A professional marketing website for Ede Marketing, built with React, Vite, Tailwind CSS, and Flowbite React.

## Live Preview

🔗 [https://jesse2704.github.io/WPI_EdeMarketing/](https://jesse2704.github.io/WPI_EdeMarketing/)

## Tech Stack

- **React 18** with Vite
- **Tailwind CSS** for styling
- **Flowbite React** for UI components
- **React Router** for navigation
- Custom context-based i18n (English / Dutch)

## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- 🌐 Bilingual support (English / Dutch) with language toggle
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Black & white minimalist design theme
- Sections: Hero, Value Props, Benefits, Before & After, Process, FAQ, Packages, Contact
- Smooth scrolling navigation
